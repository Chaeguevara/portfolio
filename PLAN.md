# Project Plan - OSM Terrain & City Generator

## Next Work: Automatic Contour & Building Generation
Generate 3D cityscapes using Three.js and OpenStreetMap (OSM) data.

### 1. Navigation & Search
- [ ] Implement coordinate and address-based lookup.
- [ ] Center camera and fetch OSM data for the specified region.

### 2. Visualization & Selection
- [ ] Generate 3D building blocks from GeoJSON/OSM data.
- [ ] Generate terrain contours.
- [ ] Highlight selected buildings with a **Pastel Red** distinct tone.

### 3. Fabrication Support
- [ ] Command-based export of the entire scene to **STL format** for 3D printing.

### 4. Stylization & Optimization (Phase 2)
- [ ] Modern, premium 3D aesthetic.
- [ ] Mesh optimization for clean 3D printing (watertight manifolds).
