# Three.js Portfolio

A lightweight, modular portfolio showcasing interactive 3D demos built with **Three.js** and **TypeScript**. Runs entirely client-side and deploys to GitHub Pages.

## ✨ Features

- **Interactive 3D Works**: Rotating boxes, orbital simulations, GPU picking, toon shading, and more
- **SPA Routing**: Client-side navigation without page reloads
- **Lazy Loading**: Heavy scenes load on demand
- **Responsive Design**: Works on desktop and mobile

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
